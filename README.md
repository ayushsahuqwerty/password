#  Cybersecurity Toolkit

## Python Challenge Project

### Overview

The Personal Cybersecurity Toolkit is a Python application that provides useful cybersecurity utilities through a simple menu-driven interface. It demonstrates core Python programming concepts while solving real-world security tasks.

---

## Features

✔ Password Strength Checker

✔ Secure Password Generator

✔ SHA-256 File Hash Generator

✔ MD5 Hash Generator

✔ SHA-1 Hash Generator

✔ SHA-512 Hash Generator

✔ File Hash Verification

✔ Base64 Text Encryption

✔ Base64 Text Decryption

✔ Caesar Cipher Encryption

✔ Caesar Cipher Decryption

✔ XOR Encryption

✔ XOR Decryption

✔ QR Code Generator

✔ Save Passwords

✔ Save Hash Reports

✔ File Integrity Checker
